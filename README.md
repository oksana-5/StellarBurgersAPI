# StellarBurgersAPI
## Автотесты для приложения космической бургерной Stellar Burgers.

### Автотесты для API

#### Создание пользователя:

создать уникального пользователя;

создать пользователя, который уже зарегистрирован;

создать пользователя и не заполнить одно из обязательных полей.

#### Логин пользователя:

вход под существующим пользователем;

вход с неверным логином и паролем.

#### Создание заказа:

с авторизацией;

без авторизации;

с ингредиентами;

без ингредиентов;

с неверным хешем ингредиентов.

### В проекте используются:
JDK corretto-11, JInit 4.13.2, Allure 2.15.0, Aspectj 1.9.7, javafaker 1.0.2, gson 2.13.1, lombok 1.18.38, rest-assured 5.5.6

#### Build system:
maven
